# Name to ASCII and Binary Conversion
This is a simple Python program that converts a user's name into ASCII values and then converts those values into binary (machine code format).
I created this project to understand how computers represent characters internally using numbers and binary data.

## Concepts Used

 `for` loop - to process each character one by one
 `function` - to organize the conversion process
 `ord()` function - to convert characters into ASCII values
 `bin()` function - to convert ASCII numbers into binary format

## How It Works

1. User enters a name
2. The program takes each character from the name
3. Each character is converted into its ASCII value
4. The ASCII value is converted into binary
5. The program displays the character, ASCII value, and binary representation
