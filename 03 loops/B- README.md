# Print Name Using For Loop

## Description
This program demonstrates the use of a **for loop** in Python to print each letter of the name "Kabilesh".

## Concepts Used
- for loop
- String
- print() function
- end parameter

## Program Logic
The program stores the name in a variable and uses a for loop to access each character one by one and print it.
