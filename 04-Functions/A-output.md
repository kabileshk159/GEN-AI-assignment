# User Defined Function in Python

## Description
This program demonstrates how to create and use a **user-defined function** in Python.

## Concepts Used
- Function definition (`def`)
- Function calling
- Parameters
- print() function

## Program Logic
A function named `add()` is created to perform addition of two numbers.  
The function is called by passing values as arguments, and it displays the result.
